## “黑洞”事件引发

* 4月11日中午，有网民发现，在视觉中国网站上，黑洞图片被列入视觉中国的“编辑图片”，并且标明：此图片是编辑图片，如用于商业用途，请致电或咨询客服代表。随后，视觉中国创始人柴继军在微信朋友圈中表示，“黑洞”照片属于Event Horizon Telescope组织（EHT），视觉中国通过合作伙伴获得编辑类使用授权。相关质疑声开始出现，舆情热度开始升温。
